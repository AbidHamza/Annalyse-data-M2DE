# Room 05 : Visualisation de données

## Objectif de la room

Cette room présente les techniques de visualisation de données avec matplotlib et seaborn. Vous apprendrez à choisir le type de graphique approprié et à créer des visualisations claires et informatives.

## Notions abordées

1. **Matplotlib**
   - Création de graphiques de base
   - Personnalisation (titres, labels, légendes)
   - Graphiques multiples (subplots)

2. **Seaborn**
   - Visualisations statistiques avancées
   - Styling et thèmes
   - Graphiques spécialisés (heatmaps, pair plots)

3. **Choix du graphique**
   - Graphiques adaptés au type de données
   - Graphiques adaptés à la question métier
   - Erreurs courantes à éviter

## Lien avec le métier

La visualisation est essentielle pour :
- Explorer les données visuellement
- Communiquer des résultats à des non-experts
- Identifier des patterns difficiles à voir dans les tableaux
- Convaincre avec des graphiques clairs

## Consignes de travail

1. Créez différents types de graphiques pour explorer les données
2. Choisissez le graphique le plus approprié pour chaque question métier
3. Personnalisez les graphiques pour la clarté
4. Interprétez chaque visualisation

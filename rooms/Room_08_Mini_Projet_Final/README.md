# Room 08 : Mini-projet final

## Objectif de la room

Ce mini-projet final vous permet de mettre en pratique toutes les compétences acquises lors de la formation. Vous devrez réaliser une analyse complète sur un dataset, depuis l'exploration jusqu'à la modélisation.

## Problème métier

Vous êtes consultant en analyse de données. Une entreprise vous confie un dataset et vous demande de :
1. Comprendre les données et leur qualité
2. Identifier les tendances et patterns
3. Formuler des recommandations basées sur l'analyse
4. Si possible, développer un modèle prédictif simple

## Dataset

Utilisez le dataset `indicateurs_socio_economiques.csv` disponible dans le dossier `/data`.

## Questions ouvertes

### Partie 1 : Exploration et nettoyage

1. Chargez et explorez le dataset
2. Identifiez et traitez les problèmes de qualité (valeurs manquantes, outliers)
3. Documentez vos choix de nettoyage

### Partie 2 : Analyse descriptive

1. Calculez les statistiques descriptives pour les variables pertinentes
2. Identifiez les corrélations entre variables
3. Visualisez les distributions et relations

### Partie 3 : Analyse exploratoire

1. Formulez 3 hypothèses métier (ex : relation entre PIB et espérance de vie)
2. Validez chaque hypothèse avec les données
3. Identifiez les insights principaux

### Partie 4 : Modélisation (optionnel mais recommandé)

1. Choisissez une variable cible (ex : espérance de vie)
2. Préparez les features
3. Entraînez un modèle de régression simple
4. Évaluez sa performance

### Partie 5 : Synthèse et recommandations

1. Synthétisez vos findings principaux
2. Formulez des recommandations actionnables
3. Préparez une présentation visuelle (graphiques clés)

## Livrables attendus

1. **Notebook Jupyter complet** contenant :
   - Code commenté et structuré
   - Analyse complète
   - Visualisations pertinentes
   - Interprétations et conclusions

2. **Rapport synthétique** (2-3 pages) :
   - Résumé de l'analyse
   - Findings principaux
   - Recommandations

## Critères d'évaluation

- Qualité du code et structure du notebook
- Complétude de l'analyse (exploration, nettoyage, analyse)
- Pertinence des visualisations
- Clarté des interprétations
- Qualité de la synthèse et des recommandations
- Application correcte des concepts vus en formation

## Durée estimée

6-8 heures de travail

## Conseil

Organisez votre travail en sections claires. N'hésitez pas à documenter votre raisonnement dans les cellules Markdown.

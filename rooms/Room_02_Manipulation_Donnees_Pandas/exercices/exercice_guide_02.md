# Exercice guidé : Analyse commerciale

## Contexte métier

Vous êtes analyste commercial et vous devez analyser les ventes pour identifier les tendances et opportunités.

## Objectifs

1. Charger le dataset `ventes_commerces.csv`
2. Identifier les meilleures catégories de produits
3. Analyser les performances par région
4. Calculer des indicateurs clés

## Questions guidées

### Étape 1 : Chargement et exploration

Chargez le dataset et affichez les informations de base (nombre de lignes, colonnes, types).

### Étape 2 : Analyse par catégorie

Calculez le chiffre d'affaires total par catégorie de produit. Quelle catégorie génère le plus de revenus ?

### Étape 3 : Analyse par région

Identifiez la région avec le plus grand nombre de transactions et la région avec le chiffre d'affaires le plus élevé.

### Étape 4 : Filtrage et agrégation

Filtrez les ventes supérieures à 150 euros, puis calculez la moyenne de ces ventes par catégorie.

### Étape 5 : Analyse croisée

Créez un tableau croisé montrant le chiffre d'affaires par catégorie ET par région. Identifiez les meilleures combinaisons.

## Livrables

- Code Python commenté
- Résultats affichés pour chaque étape
- Interprétation des résultats

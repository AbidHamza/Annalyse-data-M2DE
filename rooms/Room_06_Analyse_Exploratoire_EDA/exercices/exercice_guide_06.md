# Exercice guidé : EDA complète

## Contexte métier

Menez une analyse exploratoire complète sur le dataset des ventes.

## Objectifs

1. Explorer les distributions de toutes les variables
2. Identifier les patterns temporels (évolution dans le temps)
3. Analyser les corrélations
4. Formuler 3 hypothèses métier et les valider

## Hypothèses suggérées

1. Certaines catégories se vendent mieux dans certaines régions
2. Le prix influence la quantité vendue
3. Il existe des saisonnalités dans les ventes

## Livrables

- Notebook structuré avec sections claires
- Visualisations appropriées
- Validation de chaque hypothèse
- Conclusions et recommandations

# Room 06 : Analyse exploratoire de données (EDA)

## Objectif de la room

Cette room synthétise toutes les compétences acquises pour réaliser une analyse exploratoire complète. Vous apprendrez à formuler des hypothèses et à les valider avec les données.

## Notions abordées

1. **Processus EDA complet**
   - Exploration systématique des variables
   - Identification de patterns
   - Détection d'anomalies

2. **Formulation d'hypothèses**
   - Questions métier pertinentes
   - Hypothèses testables avec les données
   - Validation par l'analyse

3. **Synthèse et communication**
   - Structurer une analyse
   - Présenter les findings
   - Recommandations basées sur les données

## Lien avec le métier

L'EDA est l'étape fondamentale avant toute modélisation ou décision. Elle permet de :
- Comprendre profondément les données
- Identifier des opportunités ou problèmes
- Guider les décisions stratégiques
- Préparer les données pour le machine learning

## Consignes de travail

1. Menez une EDA complète sur un dataset
2. Formulez au moins 3 hypothèses métier
3. Validez ou infirmez chaque hypothèse avec les données
4. Produisez un rapport synthétique de vos findings

# Room 01 : Fondamentaux Python pour l'analyse de données

## Objectif de la room

Cette première room vise à consolider vos connaissances Python essentielles pour l'analyse de données. Nous nous concentrons sur les aspects de Python qui sont systématiquement utilisés dans le traitement de données : structures de données, fonctions, et introduction à numpy.

## Notions abordées

1. **Structures de données Python**
   - Listes, dictionnaires, tuples
   - Compréhensions de listes
   - Manipulation d'itérables

2. **Fonctions et programmation fonctionnelle**
   - Définition de fonctions
   - Paramètres et valeurs par défaut
   - Fonctions lambda
   - map, filter, reduce

3. **Introduction à NumPy**
   - Création de tableaux numpy
   - Indexation et slicing
   - Opérations vectorisées
   - Broadcasting

## Lien avec le métier

En analyse de données, vous travaillerez quotidiennement avec :
- Des collections de données (listes de mesures, dictionnaires de configurations)
- Des transformations de données (appliquer une fonction à chaque élément d'une série)
- Des calculs numériques efficaces (opérations sur des tableaux de nombres)

Python offre des outils performants pour ces tâches. La maîtrise de ces fondamentaux est indispensable avant d'utiliser pandas et les bibliothèques d'analyse.

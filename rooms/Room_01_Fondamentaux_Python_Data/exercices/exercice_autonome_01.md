# Exercice autonome : Analyse de performances de vente

## Contexte métier

Un responsable commercial vous demande d'analyser les performances de vente d'une équipe. Vous devez identifier les meilleures ventes et calculer des indicateurs de performance.

## Données

```python
ventes_mensuelles = [12500, 14300, 11800, 16200, 13900, 15100, 17800, 13400, 16700, 14200, 15900, 18100]
```

## Questions

1. Calculez le chiffre d'affaires total annuel.

2. Calculez la moyenne mensuelle et l'écart-type des ventes.

3. Identifiez les mois avec des ventes supérieures à 15000 euros. Combien y en a-t-il ?

4. Calculez l'augmentation en pourcentage entre le premier et le dernier mois.

5. Créez une fonction qui prend une liste de ventes et retourne un dictionnaire contenant :
   - Le total
   - La moyenne
   - Le minimum
   - Le maximum
   - Le nombre de mois avec ventes > 15000

6. Utilisez NumPy pour recalculer les statistiques de la question 2. Comparez les résultats.

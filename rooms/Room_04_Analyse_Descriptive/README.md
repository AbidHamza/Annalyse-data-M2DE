# Room 04 : Analyse descriptive

## Objectif de la room

Cette room présente les statistiques descriptives et leur interprétation dans un contexte d'analyse de données. Vous apprendrez à calculer et interpréter les indicateurs de tendance centrale, de dispersion, et de corrélation.

## Notions abordées

1. **Statistiques descriptives univariées**
   - Mesures de tendance centrale (moyenne, médiane, mode)
   - Mesures de dispersion (écart-type, variance, quartiles)
   - Interprétation selon le contexte

2. **Distributions**
   - Comprendre la distribution d'une variable
   - Asymétrie et kurtosis
   - Importance de la distribution pour l'analyse

3. **Corrélations**
   - Coefficient de corrélation de Pearson
   - Matrice de corrélation
   - Interprétation : corrélation vs causalité

## Lien avec le métier

L'analyse descriptive est la première étape de toute analyse. Elle permet de :
- Comprendre les caractéristiques des données
- Identifier des patterns ou anomalies
- Guider les analyses ultérieures
- Communiquer des insights aux décideurs

## Consignes de travail

1. Calculez les statistiques descriptives pour les variables pertinentes
2. Interprétez chaque indicateur dans le contexte métier
3. Identifiez les variables corrélées et interprétez ces corrélations
4. Documentez vos observations

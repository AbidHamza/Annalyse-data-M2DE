# Exercice guidé : Analyse descriptive complète

## Contexte métier

Vous devez produire un rapport statistique sur les ventes pour la direction.

## Objectifs

1. Charger le dataset `ventes_commerces.csv`
2. Calculer les indicateurs clés
3. Analyser les corrélations
4. Interpréter les résultats dans un contexte métier

## Questions

### Partie 1 : Indicateurs globaux

1. Calculez le chiffre d'affaires total
2. Identifiez la vente minimale et maximale
3. Calculez la moyenne et la médiane des montants
4. Interprétez la différence entre moyenne et médiane

### Partie 2 : Analyse par catégorie

1. Calculez le CA par catégorie
2. Identifiez la catégorie la plus performante
3. Comparez les écarts-types entre catégories

### Partie 3 : Corrélations

1. Calculez la corrélation entre prix et quantité
2. Interprétez cette corrélation dans le contexte métier
3. Y a-t-il des outliers qui influencent cette corrélation ?

## Livrables

- Code commenté
- Tableau récapitulatif des indicateurs
- Interprétation métier des résultats

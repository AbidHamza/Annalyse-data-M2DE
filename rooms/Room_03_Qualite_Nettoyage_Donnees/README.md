# Room 03 : Qualité et nettoyage de données

## Objectif de la room

Cette room aborde un aspect critique de l'analyse de données : la qualité des données. Vous apprendrez à identifier et traiter les problèmes courants dans les datasets : valeurs manquantes, doublons, outliers, et typage incorrect.

## Notions abordées

1. **Détection des valeurs manquantes**
   - Identifier les valeurs manquantes
   - Comprendre les différents types de valeurs manquantes
   - Stratégies de traitement (suppression, imputation)

2. **Gestion des doublons**
   - Détecter les doublons
   - Décider quand supprimer ou conserver

3. **Détection et traitement des outliers**
   - Méthodes de détection (IQR, z-score)
   - Décisions de traitement selon le contexte

4. **Typage des données**
   - Conversion de types
   - Dates et formats spécifiques
   - Optimisation des types pour la performance

## Lien avec le métier

Dans la pratique, les données sont rarement parfaites. Avant toute analyse, vous devez :
- Vérifier la qualité des données
- Nettoyer les erreurs évidentes
- Prendre des décisions éclairées sur le traitement des anomalies

Ces compétences sont essentielles pour produire des analyses fiables.

## Consignes de travail

1. Utilisez les datasets fournis qui contiennent volontairement des problèmes à identifier
2. Testez différentes stratégies de traitement
3. Documentez vos choix de nettoyage
4. Vérifiez l'impact du nettoyage sur les analyses

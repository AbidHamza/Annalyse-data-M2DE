# Room 07 : Introduction au Machine Learning

## Objectif de la room

Cette room introduit les concepts fondamentaux du machine learning : préparation des données, séparation train/test, modèles de base (régression et classification), et évaluation.

## Notions abordées

1. **Concepts fondamentaux**
   - Features (variables explicatives) vs labels (variable cible)
   - Apprentissage supervisé vs non supervisé
   - Régression vs classification

2. **Séparation train/test**
   - Importance de la validation
   - Éviter le surapprentissage
   - Scikit-learn pour la séparation

3. **Modèles de base**
   - Régression linéaire
   - Classification avec un algorithme simple
   - Prédictions et interprétation

4. **Évaluation de performance**
   - Métriques pour la régression (RMSE, MAE)
   - Métriques pour la classification (accuracy, confusion matrix)
   - Interprétation des métriques

## Lien avec le métier

Le machine learning permet de :
- Prédire des valeurs futures
- Classifier automatiquement
- Découvrir des patterns complexes
- Automatiser des décisions

## Consignes de travail

1. Préparez les données pour le machine learning
2. Entraînez des modèles simples
3. Évaluez leur performance
4. Interprétez les résultats dans le contexte métier
